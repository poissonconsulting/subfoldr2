# Set Database Name

Sets database name option (without the extension or path).

## Usage

``` r
sbf_set_db_name(db_name = "database")
```

## Arguments

- db_name:

  A string specifying the new database name (without the extension or
  path).

## Value

An invisible string specifying the new database name (without the
extension or path).
